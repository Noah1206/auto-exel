# 사용자 매뉴얼 (한글)

**11번가 자동 주문 프로그램 v1.0**

---

## 1. 시작하기 전

### 1.1 필요한 것

- Windows 10 (1809 이상) 또는 Windows 11 PC
- **Chrome 브라우저 설치** (필수 - 120 버전 이상)
- 11번가 계정
- (선택) 샵백 계정 및 확장프로그램

### 1.2 프로그램 파일

- `11st_auto_order.exe` (단일 실행 파일, 설치 불필요)

---

## 2. 엑셀 파일 준비

### 2.1 엑셀 컬럼

다음 6개 컬럼이 **첫 행에 헤더**로 있어야 합니다:

| 컬럼명 | 예시 |
|--------|------|
| 상품링크 | https://www.11st.co.kr/products/1234567 |
| 이름 | 김철수 |
| 번호 | 010-1234-5678 |
| 통관번호 | P123456789012 |
| 주소 | 서울시 강남구 테헤란로 123 101동 202호 |
| 영문이름 | KIM CHUL SOO |

> 컬럼 순서는 상관없으며, 헤더 이름만 정확히 맞으면 됩니다.

### 2.2 유의사항

- 번호는 `010-1234-5678`, `01012345678`, `010 1234 5678` 모두 허용 (자동 정리됨)
- 통관번호는 `P` + 숫자 12자리 형식 필수
- 영문이름은 영문 대문자만 허용 (소문자 입력 시 자동 변환)

---

## 3. 첫 실행 (1회만)

1. `11st_auto_order.exe` 더블클릭하여 실행
2. 프로그램 창 상단의 `🌐 브라우저 열기` 버튼 클릭
3. **새로운 Chrome 창**이 열립니다.
4. Chrome에서 **11번가(11st.co.kr) 접속 → 로그인**
5. Chrome 웹스토어에서 **샵백(Shopback) 확장프로그램 설치**
6. 샵백에도 로그인
7. 프로그램 창은 **닫지 마세요** (로그인 상태 저장 중)

> 첫 실행이 끝나면 이후에는 로그인 유지됩니다. 크롬 창만 닫으면 됩니다.

---

## 4. 주문 진행

### 4.1 엑셀 불러오기

- 상단 `📂 엑셀 불러오기` 클릭 → 엑셀 파일 선택
- 테이블에 주문 목록이 표시됩니다.
- 원본 파일은 자동으로 백업됩니다 (`data/backups/` 폴더).

### 4.2 일괄 가격 조회 (선택)

- `💰 일괄 가격 조회` 버튼 → 모든 상품 링크를 자동 방문하여 판매가 수집
- 50건 기준 약 2-3분 소요
- 완료 후 엑셀에 판매가 자동 저장

### 4.3 주문 실행

1. 테이블에서 주문할 행을 **더블클릭**
2. Chrome이 자동으로 해당 상품 페이지 이동
3. `바로구매` 버튼 자동 클릭
4. 주문 페이지에서 받는분/번호/주소/통관번호/영문이름 자동 입력
5. **마지막 "결제하기" 버튼은 사용자가 직접 클릭** (안전장치)
6. 결제 완료 후 **주문번호가 자동 추출되어 엑셀에 저장**

### 4.4 상태 확인

| 아이콘 | 의미 |
|--------|------|
| ⏳ 대기 | 아직 주문 전 |
| 🔄 진행중 | 현재 처리 중 |
| ✅ 완료 | 주문 완료 |
| ❌ 실패 | 오류 발생 (비고 열 참고) |

### 4.5 우클릭 메뉴

테이블에서 행 우클릭:
- 🔄 재시도
- ⏭ 건너뛰기 (완료로 표시)
- 🖼 에러 스크린샷 열기 (실패한 주문의 경우)

---

## 5. 크래시 후 이어하기

- 프로그램이 예기치 않게 종료되어도 안전합니다.
- 재실행 시 이전 세션 감지 → 완료된 주문은 자동 스킵
- 매 주문마다 엑셀이 자동 저장되므로 진행 상황이 보존됩니다.

---

## 6. 자주 묻는 질문

### Q. Chrome이 열리지 않습니다.
- Chrome이 설치되어 있는지 확인하세요. 설치되어 있어도 안 열리면 `설정 → Chrome 채널`을 `chrome` 또는 `chrome-beta`로 바꿔보세요.

### Q. "로그인이 만료되었습니다" 오류가 떠요.
- 크롬 창에서 11번가에 다시 로그인한 후 주문 재시도하세요.

### Q. 캡차가 뜨면?
- 프로그램이 자동 정지됩니다. 크롬에서 직접 캡차 해결 후 다시 행을 더블클릭하세요.

### Q. 셀렉터 오류가 자주 나요.
- 11번가 페이지 구조가 변경된 것입니다. `config/selectors.yaml` 수정 또는 개발자에게 문의하세요.

### Q. 무인 자동 주문 동작 방식
- **기본 무인 모드**입니다. 약관동의/결제수단(카드)/결제하기까지 자동 클릭하고 주문번호까지 자동 추출 → 다음 행으로 진행.
- 어떤 행에서 에러(주소 검색 실패, 판매중지, 캡차 등)가 나도 그 행만 건너뛰고 자동으로 다음 상품으로 넘어갑니다.
- 안전을 위해 수동으로 결제하고 싶으면 `⚙️ 설정` → "최종 결제 버튼 자동 클릭" 체크 해제.

---

## 7. 문제 발생 시

- 로그 파일: `data/logs/YYYY-MM-DD.log`
- 에러 스크린샷: `data/screenshots/`
- 엑셀 백업: `data/backups/`

개발자에게 문의 시 위 파일들을 함께 전달해주시면 빠른 대응이 가능합니다.

---

## 8. 안전 수칙

1. **개인 사용자 편의를 위한 도구**입니다. 11번가/샵백 이용약관을 준수해주세요.
2. 프로그램의 Chrome 프로필은 기존 개인 Chrome과 분리되어 있습니다.
3. 최종 결제는 반드시 내용을 확인 후 실행하세요.
4. 대량 주문 시 서비스 정책에 저촉될 수 있으니 주의하세요.
